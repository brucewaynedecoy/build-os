# buildos-design Agent Instructions

`buildos-design` owns deterministic hand-offs from qualified findings into `system/docs/designs/`.

Read [`README.md`](README.md), [PRD 16](../../docs/prd/16-revise-toolkit-ownership-boundaries.md), the finding contract, and the system design router before changing command behavior.

Do not add discovery-run recording, finding qualification, intake conversion, source indexing, config validation, stage-mover automation, PRD generation, plan generation, work generation, or docs closeout here. Route those domains through their owning toolkit or an explicit PRD/design revision.
